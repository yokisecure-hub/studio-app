# studio-app
Google AI Studioで作ったアプリのテスト用リポジトリ

## ブロックゲーム

スマートフォン対応のテトリス風ブロックゲームです。

### 機能
- クラシックなテトリスゲームプレイ
- スマホ対応のタッチコントロール
- レスポンシブデザイン
- スコア、レベル、ライン数の追跡
- 次のブロックのプレビュー

### 遊び方

#### PC操作
- **←→キー**: ブロックを左右に移動
- **↑キー**: ブロックを回転
- **↓キー**: ブロックを高速落下
- **スペースキー**: ブロックを即座に落下

#### スマホ操作
画面下のタッチボタンで操作できます：
- **回転ボタン**: ブロックを回転
- **左右ボタン**: ブロックを移動
- **下ボタン**: ブロックを高速落下
- **ドロップボタン**: ブロックを即座に落下

### ゲーム開始
1. `index.html`をブラウザで開く
2. 「スタート」ボタンをクリック
3. ゲームを楽しむ！

### 技術スタック
- HTML5 Canvas
- CSS3 (グラデーション、レスポンシブデザイン)
- Vanilla JavaScript
- タッチイベント対応
